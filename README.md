# Fikiri Landingpage

Statische Übersichtsseite für fikiri.de – listet die verfügbaren
Lern-Apps (aktuell: BrainChain) und verlinkt auf deren jeweilige Subdomain.

Kein Build-Schritt nötig: reines HTML/CSS, direkt deploybar (z. B. über
Cloudflare Pages mit leerem Build-Command und Root als Output-Verzeichnis).

## Struktur

- `index.html` – Startseite mit App-Übersicht
- `impressum.html`, `datenschutz.html` – rechtliche Pflichtseiten
- `style.css` – Styling (Wortmarke aktuell als Text/CSS, kein Logo-Bild)
- `assets/` – OER-Logo und selbst gehostete Schriftdateien (Poppins)
